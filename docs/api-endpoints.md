# API Endpoints

## Auth

- `POST /api/v1/auth/register`
- `POST /api/v1/auth/login`
- `GET /api/v1/auth/me`

## Users

- `GET /api/v1/users/me`
- `PATCH /api/v1/users/me`

## Planner

- `GET /api/v1/planner`
- `POST /api/v1/planner`
- `PATCH /api/v1/planner/:planId`
- `DELETE /api/v1/planner/:planId`
- `GET /api/v1/planner/tasks`
- `POST /api/v1/planner/tasks`
- `PATCH /api/v1/planner/tasks/:taskId`
- `DELETE /api/v1/planner/tasks/:taskId`

## Analytics

- `GET /api/v1/analytics/overview`
- `POST /api/v1/analytics/records`

## Predictions

- `GET /api/v1/predictions/me`
