# StudySync AI

StudySync AI is a MERN stack smart study planner with performance analytics and regression-based score prediction.

## Apps

- `server/`: Express REST API with JWT auth and modular MVC structure
- `client/`: React + Vite frontend with scalable feature-based organization
- `docs/`: API, schema, and architecture notes

## Quick Start

1. Install dependencies in `server/` and `client/`
2. Copy `.env.example` to `.env` in each app
3. Start MongoDB locally or with `docker-compose up -d`
4. Run the backend and frontend dev servers
